React Native 串接api 作業

作業介紹

這作業利用串接- 農業開放資料平台
內容是串接資料為: 動物收容所名稱、動物品種以及點開後相關動物訊

動物就像我們的家人，選擇領養，給寵物家溫暖與愛
Animal頁面可以看到貓狗列表，點進去可以看到更詳細的資訊
Food頁面可以看到貓狗的糧食，點進去可看見食品產地、成分以及如何食用保存等等資訊

開發使用環境/工具

  • Node.js 版本需為 v16 以上
  • Bootstrap v5.3.3
  • React Native
  • Expo
