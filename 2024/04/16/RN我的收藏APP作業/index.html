React Native 我的收藏APP 作業

作業介紹

這作業利用串接- 農業開放資料平台
內容是串接資料為: 農產品名稱、農會相關資訊

開發使用環境/工具

  • Node.js 版本需為 v16 以上
  • Bootstrap v5.3.3
  • React Native
  • Expo

作業內有的功能

  • 在列表左方可勾選喜歡的農產品
  • 列表右方的箭頭點擊後可以觀看產品詳細資訊
  • 另外，Cart頁面可看見勾選過的農產品紀錄
